# cariste-runner
Cariste Runner 2D 🚜💨 Un Endless Runner fun et addictif développé en PWA avec HTML, CSS et JavaScript vanilla ! Prenez les commandes d'un chariot élévateur et esquivez les palettes en sautant au bon moment. Le but : tenir le plus longtemps possible, battre votre score et éviter l'accident de travail. Installable et fluide !
